### GraphQL Server

#### apolloServer

```
npm install apollo-server graphql
npm install --save-dev nodemon 
npx nodemon index.js
```

#### expressGraphql
https://graphql.org/graphql-js/running-an-express-graphql-server/

```
npm install express express-graphql graphql --save
node server.js
```
